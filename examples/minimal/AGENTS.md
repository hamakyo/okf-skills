# Codex Instructions

This repository uses OKF for project knowledge and Skills for repeatable workflows.

## Before Work

- Read `README.md` if present.
- Read `okf/index.md` before implementation work.
- Read the relevant `skills/*/SKILL.md` before using a Skill.
- If editing OKF, also read `okf/log.md`.

## Rules

- Keep project facts in `okf/`.
- Keep agent workflow steps in `skills/`.
- Prefer small, focused diffs.
- Report tests or validation in the final response.
- Do not add secrets, credentials, private URLs, or personal data.
