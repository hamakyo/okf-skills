# Claude Code Instructions

This repository uses OKF as the project knowledge layer and Skills as task workflows.

## Usage

- Read `okf/index.md` before feature work.
- Use `skills/implement-feature/SKILL.md` for behavior changes.
- Use `skills/update-okf/SKILL.md` after changes that affect durable project knowledge.

## Checklist

- [ ] Relevant OKF was read.
- [ ] The selected Skill was followed.
- [ ] Changes are focused.
- [ ] Validation is reported.
- [ ] OKF was updated when behavior, architecture, data, domain, or playbooks changed.
